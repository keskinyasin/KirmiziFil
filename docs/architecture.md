# Kırmızı Fil

## Proje Mimarisi

Kırmızı Fil; Repair Cafe, ikinci el market, workshop, topluluk, blog ve etkinlik yönetimini tek platform altında birleştiren modern bir Progressive Web App (PWA) olarak geliştirilmektedir.

Bu proje uzun yıllar geliştirilebilecek şekilde ölçeklenebilir tasarlanacaktır.
