# Kırmızı Fil

Kırmızı Fil; onarım kültürünü, döngüsel ekonomiyi, ikinci el kullanımını, topluluk dayanışmasını ve sosyal girişimciliği bir araya getiren modern bir dijital platformdur.

Bu proje; Repair Cafe, Workshop, 2. El Market, Kafe, Blog, Etkinlikler ve Topluluk modüllerinden oluşan ölçeklenebilir bir ASP.NET Core MVC uygulamasıdır.

---

## Vizyon

Kırmızı Fil'i yalnızca bir web sitesi değil;

- topluluk platformu,
- etkinlik yönetim sistemi,
- ikinci el pazaryeri,
- içerik yönetim sistemi,
- Progressive Web App (PWA)

olarak geliştirmek.
